# 🔐 Auth API - Node.js + Express + MongoDB

Une API simple d'authentification avec enregistrement, login, JWT, gestion des utilisateurs et rôles (admin).

## 🚀 Stack

- Node.js
- Express
- MongoDB + Mongoose
- JSON Web Token (JWT)
- bcryptjs
- dotenv

---

## 📦 Installation

```bash
git clone https://github.com/ton-pseudo/auth-api.git
cd auth-api
npm install
